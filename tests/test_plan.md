# Notas de pruebas rápidas

## Prueba manual mínima

1. Ejecutar `python3 src/main.py`.
2. Probar opciones 1, 2 y 3 con texto válido.
3. Probar opción 4 para confirmar que todo se lista.
4. Ingresar opción inválida (`0` o `x`) y validar advertencia.
5. Elegir opción 5 y confirmar salida limpia.

## Resultado esperado

- Flujo completo sin excepciones.
- Mensajes claros para usuario.
- Comportamiento consistente entre ejecuciones.
