# Casos de uso

## Perfil de usuario

- Estudiante de preparatoria o universidad.
- Prefiere herramientas ligeras.
- No quiere perder tiempo configurando software complejo.

## Escenario 1: Captura de nota rápida

**Contexto:** durante una sesión de estudio de cálculo.

1. El usuario ejecuta el asistente.
2. Selecciona “Agregar nota”.
3. Escribe: “Revisar regla de la cadena con 3 ejercicios”.
4. El sistema confirma guardado.

**Resultado esperado:** la nota aparece en el resumen de sesión.

## Escenario 2: Registro de preguntas para repaso

**Contexto:** al terminar un tema, el usuario quiere medir comprensión.

1. Selecciona “Agregar pregunta”.
2. Escribe: “¿Cuál es la diferencia entre mitosis y meiosis?”.
3. El sistema confirma guardado.

**Resultado esperado:** la pregunta queda lista para revisarse después.

## Escenario 3: Recordatorio breve

**Contexto:** el usuario necesita no olvidar una tarea.

1. Selecciona “Agregar recordatorio”.
2. Escribe: “Mañana resolver guía 2 antes de las 8 pm”.

**Resultado esperado:** el recordatorio aparece en el resumen.

## Escenario 4: Cierre de sesión

1. El usuario selecciona “Ver resumen”.
2. Revisa notas, preguntas y recordatorios.
3. Selecciona “Salir”.

**Resultado esperado:** cierre limpio con mensaje de despedida.
